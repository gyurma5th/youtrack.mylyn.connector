    Eclipse plugin connector for YouTrack bugtracker using Mylyn

    Project maintained by Amarchuk Hosted on GitHub Pages — Theme by mattgraham

    Requirements

    Eclipse IDE version >= 4.2 with installed Mylyn (http://wiki.eclipse.org/Mylyn/FAQ#Installation).

    Installation

    You can install the editor either using the "update site" method or manually by copying the associated jar files; see below sections for details.

    The update site method allows for updating the plugin within Eclipse IDE:

    Help -> Check for Updates

    Installing using update site

    1. Start Eclipse, then select Help -> Install New Software.
    2. Click Add, in the top-right corner. In the Add Repository dialog that appears, enter "YouTrack Mylyn Connector" for the Name and the following URL for the Location: http://jetbrains.github.io/youtrack.mylyn.connector/updatesite/update/
    3. Click OK.
    4. In the Available Software dialog, select the checkbox next to YouTrack Mylyn Connector and click Next. If content not found, deselect Group items by category.
    5. In the next window, you'll see a list of the tools to be downloaded. Click Next.
    6. Read and accept the license agreements, then click Finish.
    7. When the installation completes, restart Eclipse.
    8. See usage for more information about using the plugin.

    Installing manually

    Download the jar file of the feature version wanted from https://github.com/JetBrains/youtrack.mylyn.connector/tree/gh-pages/updatesite/update/features.

    Then download 3 jars of the plugins with the same version as the feature from https://github.com/JetBrains/youtrack.mylyn.connector/tree/gh-pages/updatesite/update/plugins/.

    Close your Eclipse and copy the feature jar file to your Eclipse's features directory and the second jars to your Eclipse's plugins directory. You can find the feature and plugin directories under your Eclipse installation directory, eg. ~/eclipse/features and ~/eclipse/plugins.

    Restart your Eclipse and you can now works with connector.

    Uninstallation

    Uninstallation when installed using update-site

    1. Start Eclipse, select Help -> About Eclipse.
    2. Click Installation Details.
    3. In the pre-selected Installed Software tab, select the line with "YouTrack Mylyn Connector".
    4. Click Uninstall...
    5. In the Uninstal Details dialog click Finish.
    6. Restart Eclipse to complete uninstallation.

    Uninstalling manually installed plugin

    Close your Eclipse and go to your Eclipse plugin directory, usually something like ~/eclipse/plugins and delete or move 3 plugin files. The plugin names are following:

    com.jetbrains.mylyn.yt.core_[version].jar,
com.jetbrains.mylyn.yt.ui_[version].jar,
com.jetbrains.youtrack.javarest_[version].jar

    Also delete feature jar from ~/eclipse/features and all mentions in artifacts.xml (do it on your own risk).

    Next time you start Eclipse, YouTrack Mylyn Connector has been uninstalled.
